# Modelo-Preditivo-para-E-commerce

Projeto em **Python** que cria um **modelo preditivo** para identificar a probabilidade de um cliente realizar uma compra em um e-commerce, com base em dados comportamentais.

---

* Python 3
* Pandas
* Scikit-learn
* Matplotlib

---

## O que o modelo faz

* Cria um conjunto de dados fictício de clientes
* Separa dados de treino e teste
* Treina um modelo **Decision Tree Classifier**
* Avalia o desempenho com **acurácia** e **relatório de classificação**
* Exibe a árvore de decisão graficamente
